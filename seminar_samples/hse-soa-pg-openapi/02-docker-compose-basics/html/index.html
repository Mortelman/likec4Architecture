Docker Compose — Пример 02

Сервисы запущены!

Это страница от nginx — статический веб-сервер.

Перейди на http://localhost:8081 чтобы увидеть сервис whoami.

Что здесь происходит?

  • Docker Compose запустил два контейнера из одного файла
  • nginx отдаёт эту HTML страницу с хоста (bind mount)
  • whoami показывает заголовки запроса и информацию о контейнере
